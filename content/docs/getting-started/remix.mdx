# Remix Setup

This guide will help you integrate Flint UI into your Remix application.

## Prerequisites

- Remix 2+
- React 18+
- Node.js 16+

## Setup

### 1. Initialize Tailwind CSS

If you haven't already, set up Tailwind CSS in your Remix app:

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### 2. Configure Tailwind CSS

Update your `tailwind.config.ts`:

```typescript title="tailwind.config.ts"
import type { Config } from 'tailwindcss';
import flintPreset from '@flint-org/ui/preset';

export default {
  presets: [flintPreset],
  content: [
    './app/**/*.{js,jsx,ts,tsx}',
    './node_modules/@flint-org/ui/dist/**/*.js',
  ],
  darkMode: ['class'],
} satisfies Config;
```

### 3. Import Styles

Add Flint UI styles to your root route:

```tsx title="app/root.tsx"
import type { LinksFunction } from '@remix-run/node';
import {
  Links,
  LiveReload,
  Meta,
  Outlet,
  Scripts,
  ScrollRestoration,
} from '@remix-run/react';

import flintStyles from '@flint-org/ui/styles?url';
import stylesheet from '~/tailwind.css?url';

export const links: LinksFunction = () => [
  { rel: 'stylesheet', href: flintStyles },
  { rel: 'stylesheet', href: stylesheet },
];

export default function App() {
  return (
    <html lang="en">
      <head>
        <meta charSet="utf-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1" />
        <Meta />
        <Links />
      </head>
      <body>
        <Outlet />
        <ScrollRestoration />
        <Scripts />
        <LiveReload />
      </body>
    </html>
  );
}
```

### 4. Use Components

```tsx title="app/routes/_index.tsx"
import { Button, Card, CardHeader, CardTitle, CardContent } from '@flint-org/ui';

export default function Index() {
  return (
    <div className="container mx-auto p-8">
      <Card>
        <CardHeader>
          <CardTitle>Welcome to Flint UI</CardTitle>
        </CardHeader>
        <CardContent>
          <p>Get started with beautiful, accessible components.</p>
          <Button className="mt-4">Get Started</Button>
        </CardContent>
      </Card>
    </div>
  );
}
```

## Dark Mode Setup

Install `next-themes` (works with Remix!):

```bash
npm install next-themes
```

Create a theme provider:

```tsx title="app/components/theme-provider.tsx"
import { ThemeProvider as NextThemesProvider } from 'next-themes';
import type { ReactNode } from 'react';

export function ThemeProvider({ children }: { children: ReactNode }) {
  return (
    <NextThemesProvider attribute="class" defaultTheme="system" enableSystem>
      {children}
    </NextThemesProvider>
  );
}
```

Update your root route:

```tsx title="app/root.tsx"
import { ThemeProvider } from '~/components/theme-provider';

export default function App() {
  return (
    <html lang="en" suppressHydrationWarning>
      <head>
        {/* ... */}
      </head>
      <body>
        <ThemeProvider>
          <Outlet />
        </ThemeProvider>
        <ScrollRestoration />
        <Scripts />
        <LiveReload />
      </body>
    </html>
  );
}
```

Use the theme switcher:

```tsx title="app/components/header.tsx"
import { ThemeSwitcher } from '@flint-org/ui';

export function Header() {
  return (
    <header className="flex justify-between items-center p-4 border-b">
      <h1 className="text-xl font-bold">My App</h1>
      <ThemeSwitcher />
    </header>
  );
}
```

## Client-Only Components

Some Flint UI components use browser APIs and need to be rendered client-side only:

```tsx title="app/routes/_index.tsx"
import { ClientOnly } from 'remix-utils/client-only';
import { ThemeSwitcher } from '@flint-org/ui';

export default function Index() {
  return (
    <div>
      <ClientOnly fallback={<div>Loading...</div>}>
        {() => <ThemeSwitcher />}
      </ClientOnly>
    </div>
  );
}
```

## Server-Side Rendering

Flint UI components are SSR-compatible. However, some features like animations require client-side JavaScript:

```tsx
import { useEffect, useState } from 'react';

export function ClientOnlyComponent() {
  const [isClient, setIsClient] = useState(false);

  useEffect(() => {
    setIsClient(true);
  }, []);

  if (!isClient) {
    return <div>Loading...</div>;
  }

  return <AnimatedComponent />;
}
```

## Optimizing Bundle Size

Remix automatically code-splits by route. Import components normally:

```tsx
import { Button, Card, Input } from '@flint-org/ui';
```

## Troubleshooting

### Hydration mismatches

Ensure theme provider has `suppressHydrationWarning` on the `<html>` tag.

### Styles not loading

Make sure `flintStyles` is included in your `links` export and comes before your own styles.

### Module not found

Run `npm install` to ensure all dependencies are installed, and restart your dev server.
